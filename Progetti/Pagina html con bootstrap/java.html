La nascita

La storia di Java inizia nel 1991 quando James Gosling, Mike Sheridan e Patrick Naughton, tre ingegneri di Sun Microsystems, decisero di sviluppare un linguaggio di programmazione per dispositivi intelligenti.
Inizialmente, il progetto era chiamato "Oak" a causa di un albero di quercia che si trovava fuori dall'ufficio di Gosling.
Tuttavia, successivamente il nome fu cambiato in Java a causa di problemi legali legati al marchio Oak.

Il Boom dell'Internet e Java Applets

Nel 1995, Java fu rilasciato al pubblico e divenne subito popolare grazie alla sua portabilità e alla possibilità di eseguire il codice su diverse piattaforme senza la necessità di essere riadattato. Questa caratteristica rese Java ideale per lo sviluppo di applicazioni Web, che in quel periodo stavano iniziando a guadagnare popolarità. Un elemento chiave di questa era fu l'introduzione degli "applet Java". Gli sviluppatori potevano incorporare piccoli programmi Java direttamente nelle pagine web, consentendo l'esecuzione di applicazioni interattive all'interno di un browser. Questa caratteristica contribuì notevolmente all'espansione di Java.

Java 2 e l'Enterprise

Con l'evoluzione delle esigenze di sviluppo aziendale, Sun Microsystems rilasciò Java 2 nel 1998.
Questa versione introduceva molte nuove funzionalità, tra cui il supporto per le librerie di sviluppo Enterprise Edition (J2EE).
Ciò ha permesso agli sviluppatori di creare applicazioni più complesse e scalabili per affrontare le sfide del mondo aziendale.

La Rivoluzione di Android

Uno degli sviluppi più significativi nella storia di Java è stata l'adozione di Android, il sistema operativo mobile di Google, che ha iniziato a utilizzare Java come linguaggio di programmazione principale per le app Android. Questo ha ampliato ulteriormente la portata di Java, portandolo a essere uno dei linguaggi di programmazione più diffusi al mondo.

Oracle e Java

Nel 2010, Oracle acquisì Sun Microsystems, diventando il custode ufficiale di Java. Questa transizione portò a diverse controversie legali, in particolare la causa legale tra Oracle e Google riguardo all'utilizzo di Java in Android. La Corte Suprema degli Stati Uniti emise una sentenza a favore di Google nel 2021, stabilendo che l'uso di Java in Android costituiva un uso lecito.

Java Oggi

Oggi, Java è ancora ampiamente utilizzato in una varietà di contesti, dallo sviluppo di applicazioni enterprise alle app Android. La sua portabilità, affidabilità e ampio ecosistema di librerie e framework continuano a renderlo una scelta popolare tra gli sviluppatori di tutto il mondo.
In conclusione, la storia di Java è una storia di adattamento e evoluzione. Da un linguaggio progettato per dispositivi intelligenti, Java è diventato uno dei pilastri fondamentali dello sviluppo software, alimentando l'era di Internet e continuando a influenzare il panorama tecnologico odierno.

Caratteristiche
Portabilità:

"Write Once, Run Anywhere" grazie alla Java Virtual Machine (JVM).

Orientato agli oggetti:

Utilizza il paradigma di programmazione orientato agli oggetti.

Sicurezza:

Integrata con meccanismi di sicurezza per prevenire violazioni.

Gestione automatica della memoria:

Grazie al Garbage Collector.

Thread e concorrenza:

Supporto integrato per la programmazione concorrente.

Librerie standard (API):

Vasta libreria standard per compiti comuni.

Interfaccia grafica:

Librerie per la creazione di GUI.

Comunità di sviluppatori:

Ampia e attiva comunità di sviluppatori.

Sviluppo multipiattaforma:

Utilizzato in applicazioni desktop, web, mobile e embedded.

Versioni LTS:

Introduzione del modello LTS per garantire il supporto a lungo termine.
